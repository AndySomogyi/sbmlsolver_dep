In this model, an event uses trigger-time values, which makes a difference to the results.

The model contains:

  • 2 parameters (x, y)

There are 2 events:

Event Trigger Priority Event Assignments
E1 time > 3.5 10 x = 7
E2 time > 3.5 5 y = x

The initial conditions are as follows:

Value Constant
Initial value of parameter x 3 variable
Initial value of parameter y 0 variable

Note: The test data for this model was generated from an analytical solution of the system of equations.
