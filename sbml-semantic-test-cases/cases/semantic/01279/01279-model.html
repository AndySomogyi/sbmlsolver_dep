In this model, the 'implies' MathML symbol is used in assignment rules. 'Implies' means 'if A is true, then B is also true', or '(not A) or B'. With the automatic translation (in l3v2) of boolean values to numeric values, this means that p1 is always 1 (true); p2 becomes 0 (false) halfway through the run; p3 becomes 0 only from 2.5
