In this test, a single reaction has multiple reactants that all point to the same species, each of which has an assigned value. This is equivalent to a reaction where all of the species references that point to the same species are collapsed into a single species reference with a stoichiometry equal to the sum of the stoichiometries of the original species references. Partway through the simulation, the stoichiometries change due to an event.

The model contains:

  • 1 species (A)
  • 1 compartment (C)
  • 2 species references (A1sr, A1sr2)

There is one reaction:

Reaction Rate
J0: A1sr A + A1sr2 A → −1

Note: the following stoichiometries are set separately: A1sr, A1sr2

There is one event:

Event Trigger Event Assignments
_E0 time ≥ 5 A1sr = 1
A1sr2 = 1

The initial conditions are as follows:

Value Constant
Initial concentration of species A 1 variable
Initial volume of compartment 'C' 1 constant
Initial value of species reference 'A1_sr' 2 constant
Initial value of species reference 'A1_sr2' 1 constant
